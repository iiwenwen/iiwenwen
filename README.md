## 一封短笺

你好，我是 **Syaoran**。

住在福建宁德，一座靠近山海的城市。

我写代码，也写那些不适合写进代码里的事。<br>
写读过的书、看过的电影、偶然落下的雨，<br>
也写一些尚未抵达的人，和已经走远的时光。

我喜欢缓慢但长久的东西：<br>
一座独立网站，一封多年以后才被读到的信，<br>
一首没有写完的诗，和一个仍在成为自己的人。

> 不要哀求，学会争取。若能如此，终有所得。
